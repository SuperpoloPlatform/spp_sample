简介：
    该Demo实现了视角的调节和改变，以及鼠标点击对象弹出对话框。
	
详细介绍：
     1.绑定键位及鼠标操作与相应的动作。
     2.根据不同的键位进行改变视角，调节视角。
     3.鼠标点击不同盒子，产生的对话框内容不同。	 
	 
展示简介： 
     1.该场景为无重力模式。
	 2.2个盒子，分比为player和monster。
	 
功能展示：
	1.通过脚本或者DOS命令，用spp.exe引擎运行start.js文件。
	2.待程序运行起来后，弹出对话框“使用鼠标左键单击某个盒子”。
	3.用鼠标左键分别点击两个盒子，本别会弹出对话框“player”和“monster”。
	
重点实现代码：
	1.start.js中player键位的绑定与消息的传递、执行，以及详细实现。         29-179行
	2.start.js中monster与mesh的绑定与消息的传递、执行，以及详细实现。      182-192行
	
注意问题：
	1、不能缺少的几个PC"pclinearmovement"“pcmeshselect”“pccommandinput”。
	2、键位的按下和抬起是两个动作，需要两个消息函数。
	3、start中包含了效果和对象，是一个集合。由于没有request(库名.js)，所以先读取factory之后再创建对象。
